### Process
1. 각자 읽고 싶은 도서를 1권 고른다.
2. 약 한 달간 해당 도서를 읽고, 요약을 작성한다.
3. 작성 된 요약을 모여서 읽는다.
4. 서로의 요약본에서 질문이 있거나, 나눌 감상이 있으면 공유한다.
5. 해당 내용을 기록하고 아카이빙한다.
6. 위의 단계를 모임이 종료될때 까지 반복한다.
